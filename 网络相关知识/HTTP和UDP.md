## HTTP 和 UDP

TCP 是面向链接的，而 UDP 是面向无连接的。

TCP 仅支持单播传输，UDP 提供了单播，多播，广播的功能。

TCP 的三次握手保证了连接的可靠性; UDP 是无连接的、不可靠的一种数据传输协议，首先不可靠性体现在无连接上，通信都不需要建立连接，对接收到的数据也不发送确认信号，发送端不知道数据是否会正确接收。
UDP的头部开销比TCP的更小，数据传输速率更高，实时性更好。
